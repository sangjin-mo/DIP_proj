# 임베디드 시스템의 종류

## 1. 수행 목표

본 문서는 일상생활 속에서 사용하고 있는 임베디드 시스템을 인식하고, 임베디드 시스템의 정의, 구성 요소, 종류, 사례를 정리하기 위해 작성하였다.
또한 하루 동안 사용하는 임베디드 시스템을 시간순으로 나열하고, 각 장비가 생활에 미치는 영향을 분석한다.
마지막으로 아직 존재하지 않지만 일상에 필요할 것으로 생각되는 새로운 임베디드 장비를 구상한다.

---

## 2. 임베디드 시스템의 정의

임베디드 시스템(Embedded System)은 특정한 기능을 수행하기 위해 하드웨어와 소프트웨어가 결합된 전자 시스템이다.
일반적인 PC처럼 여러 목적을 수행하기보다는, 정해진 목적을 안정적으로 수행하도록 설계된다.

예를 들어 스마트폰, 자동차 제어기, 전자레인지, 인터넷 공유기, 스마트 도어락, 산업용 로봇, 의료기기 등이 임베디드 시스템에 해당한다.

### 2.1 임베디드 시스템의 특징

| 특징             | 설명                               |
| -------------- | -------------------------------- |
| 특정 목적 수행       | 정해진 기능을 반복적으로 수행한다.              |
| 하드웨어와 소프트웨어 결합 | 센서, 제어기, 펌웨어 등이 함께 동작한다.         |
| 실시간성           | 입력에 대해 정해진 시간 안에 반응해야 하는 경우가 많다. |
| 안정성            | 장시간 동작해도 오류가 적어야 한다.             |
| 제한된 자원         | 메모리, 전력, 연산 성능이 제한되는 경우가 많다.     |

---

## 3. 임베디드 시스템을 구성하는 최소 조건

임베디드 시스템이 되기 위해서는 최소한 다음과 같은 요소가 필요하다.

| 구성 요소            | 역할                                  |
| ---------------- | ----------------------------------- |
| 마이크로컨트롤러 또는 프로세서 | 명령을 처리하고 전체 시스템을 제어한다.              |
| 입력 장치            | 센서, 버튼, 카메라, 통신 입력 등을 통해 정보를 받는다.   |
| 출력 장치            | 모터, 디스플레이, LED, 스피커, 통신 출력 등을 제어한다. |
| 소프트웨어            | 기능 수행을 위한 펌웨어 또는 운영체제가 필요하다.        |
| 전원               | 배터리, 어댑터, 차량 전원 등을 통해 동작한다.         |
| 통신 기능            | 필요 시 다른 장치와 데이터를 주고받는다.             |

임베디드 시스템은 기본적으로 다음과 같은 흐름으로 동작한다.

```text
입력 장치 → 제어기/프로세서 → 출력 장치
 센서값       판단·연산        모터·화면·통신
```

즉, 임베디드 시스템은 단순한 전자회로가 아니라 **입력, 처리, 출력의 흐름을 가진 제어 시스템**이라고 볼 수 있다.

## 4. 펌웨어, RTOS, 임베디드 OS의 용도

| 구분      | 설명                              | 주요 용도                     |
| ------- | ------------------------------- | ------------------------- |
| 펌웨어     | 하드웨어 내부에 저장되어 장치를 직접 제어하는 소프트웨어 | 전자레인지, 도어락, 센서 모듈, 모터 제어기 |
| RTOS    | 정해진 시간 안에 작업을 처리하기 위한 실시간 운영체제  | 자동차 ECU, 로봇 제어기, 산업용 장비   |
| 임베디드 OS | 임베디드 장치에서 여러 기능을 실행하기 위한 운영체제   | 스마트폰, 스마트 TV, 네트워크 장비     |

### 4.1 펌웨어

펌웨어는 장치 내부에 저장되어 하드웨어를 직접 제어하는 프로그램이다.
예를 들어 전자레인지의 버튼 입력 처리, 타이머 동작, 출력 제어 등이 펌웨어로 구현된다.

### 4.2 RTOS

RTOS는 Real-Time Operating System의 약자로, 실시간 처리를 위한 운영체제이다.
자동차, 로봇, 의료기기, 산업용 장비처럼 정해진 시간 안에 정확히 반응해야 하는 시스템에서 사용된다.

### 4.3 임베디드 OS

임베디드 OS는 특정 장비에 맞게 최적화된 운영체제이다.
스마트폰의 Android, 스마트 TV 운영체제, Linux 기반 네트워크 장비 OS 등이 이에 해당한다.

## 5. 하루 동안 사용하는 임베디드 시스템

아침에 기상한 시점부터 하루 동안 사용하는 임베디드 시스템을 시간순으로 정리하면 다음과 같다.

| 시간대   | 사용하는 임베디드 시스템 | 사용 목적            | 나에게 미치는 영향               |
| ----- | ------------- | ---------------- | ------------------------ |
| 기상 직후 | 스마트폰 알람       | 기상 시간 알림         | 하루 일정을 시작하게 해준다.         |
| 아침    | 스마트폰          | 메시지, 날씨, 일정 확인   | 정보 확인과 일정 관리에 사용된다.      |
| 아침    | 전자레인지         | 음식 데우기           | 식사 준비 시간을 줄여준다.          |
| 외출 전  | 스마트 도어락       | 문 개폐 및 보안        | 열쇠 없이 출입할 수 있고 보안성을 높인다. |
| 이동 중  | 교통카드 단말기      | 버스, 지하철 요금 결제    | 대중교통 이용을 편리하게 한다.        |
| 이동 중  | 버스 위치 안내 시스템  | 도착 시간 확인         | 이동 계획을 세우는 데 도움을 준다.     |
| 학교/회사 | 출입통제 시스템      | 출입 인증            | 보안 관리와 출입 기록에 사용된다.      |
| 작업 중  | 노트북/컴퓨터       | 문서 작성, 코딩, 자료 검색 | 학습과 업무 수행의 핵심 장비이다.      |
| 작업 중  | 인터넷 공유기       | 네트워크 연결          | 검색, 협업, 자료 전송이 가능하게 한다.  |
| 저녁    | 세탁기           | 세탁 자동화           | 가사 노동을 줄여준다.             |
| 저녁    | 스마트 TV 또는 태블릿 | 영상 시청, 정보 확인     | 여가 활동과 정보 확인에 사용된다.      |
| 취침 전  | 스마트폰          | 알람 설정, 일정 확인     | 다음 날 일정을 준비하게 해준다.       |

일상생활에서 사용하는 대부분의 전자제품은 특정 기능을 수행하기 위한 임베디드 시스템을 포함하고 있다.

## 6. 스마트폰 장애 또는 분실 시 영향

스마트폰은 통신, 결제, 인증, 일정 관리, 교통, 정보 검색 기능을 모두 포함하는 장비이다.
스마트폰을 잃어버리거나 고장이 나면 다음과 같은 문제가 발생할 수 있다.

| 영향       | 설명                                 |
| -------- | ---------------------------------- |
| 연락 불가    | 전화, 문자, 메신저 사용이 어려워진다.             |
| 인증 문제    | 금융 앱, 학교/회사 서비스, 2단계 인증 사용이 어려워진다. |
| 결제 불편    | 간편결제, 모바일 교통카드 사용이 제한된다.           |
| 일정 관리 문제 | 알람, 캘린더, 메모 확인이 어려워진다.             |
| 위치 확인 불가 | 지도, 대중교통 정보 이용이 어려워진다.             |
| 심리적 불안   | 개인정보 유출과 일정 차질에 대한 불안이 발생한다.       |

따라서 스마트폰은 단순한 통신 장비가 아니라 개인 생활의 중심이 되는 임베디드 시스템이라고 볼 수 있다.

## 7. 가장 많이 사용하는 임베디드 장비 순위

본인이 사용하는 시간을 기준으로 생각하면 다음과 같다.

| 순위 | 장비          | 이유                                    |
| -- | ----------- | ------------------------------------- |
| 1위 | 스마트폰        | 통신, 검색, 결제, 인증, 일정 관리 등 사용 시간이 가장 길다. |
| 2위 | 노트북/컴퓨터     | 학습, 문서 작성, 코딩, 자료 검색에 많이 사용한다.        |
| 3위 | 인터넷 공유기     | 스마트폰과 컴퓨터의 네트워크 연결에 항상 사용된다.          |
| 4위 | 교통카드/교통 단말기 | 이동할 때 반복적으로 사용한다.                     |
| 5위 | 가전제품        | 전자레인지, 세탁기, 냉장고 등 생활 편의를 제공한다.        |

스마트폰 다음으로 많이 사용하는 장비는 개인 기준으로 **노트북 또는 컴퓨터**라고 볼 수 있다.

---

## 8. 많은 사람들이 이용하는 임베디드 장비

많은 사람들이 이용하는 임베디드 장비를 생각하면 다음과 같다.

| 순위 | 장비              | 이유                                |
| -- | --------------- | --------------------------------- |
| 1위 | 스마트폰            | 거의 모든 사람이 통신과 정보 검색에 사용한다.        |
| 2위 | 인터넷 공유기/네트워크 장비 | 가정, 학교, 회사, 공공장소에서 항상 사용된다.       |
| 3위 | 교통카드 단말기        | 대중교통 이용 시 많은 사람이 사용한다.            |
| 4위 | 자동차 전장 시스템      | 자동차 이용자에게 필수적인 시스템이다.             |
| 5위 | 가전제품            | 냉장고, 세탁기, 전자레인지 등 대부분의 가정에서 사용된다. |

스마트폰 다음으로는 **인터넷 공유기와 네트워크 장비**가 많은 사람들이 이용하는 임베디드 시스템이라고 생각된다.

## 9. 본인에게 가장 중요한 임베디드 장비

본인에게 가장 중요한 임베디드 장비는 **스마트폰**이다.

### 9.1 스마트폰이 중요한 이유

1. 연락, 메신저, 이메일 확인이 가능하다.
2. 알람, 일정, 메모 기능을 통해 하루 일정을 관리한다.
3. 지도, 대중교통, 간편결제 기능을 사용할 수 있다.
4. 학교, 회사, 금융 서비스의 인증 수단으로 사용된다.
5. 사진, 파일, 계정 정보 등 개인 데이터가 저장되어 있다.

스마트폰은 단순한 전자기기가 아니라 생활, 학습, 이동, 금융, 인증 기능을 모두 포함하는 개인용 임베디드 플랫폼이다.

## 10. 임베디드 시스템의 종류

임베디드 시스템은 사용 분야에 따라 다양하게 분류할 수 있다.

| 분야     | 예시                                       |
| ------ | ---------------------------------------- |
| 차량용    | 자동차 ECU, ABS, 에어백 제어기, ADAS, 내비게이션, 블랙박스 |
| 산업용    | PLC, 산업용 로봇, 공장 자동화 장비, 센서 제어 장치         |
| 의료용    | 심전도 측정기, 혈압계, 인슐린 펌프, 의료 영상 장비           |
| 가정용    | 전자레인지, 세탁기, 냉장고, 스마트 TV, 로봇청소기           |
| 인터넷 장비 | 공유기, 스위치, 모뎀, 기지국 장비                     |
| 로봇     | 모바일 로봇, 협동로봇, 물류 로봇, 서비스 로봇              |
| 우주항공   | 위성 제어기, 항공기 제어 시스템, 드론 비행 제어기            |
| 보안 장비  | CCTV, 스마트 도어락, 출입통제 시스템                  |
| 웨어러블   | 스마트워치, 헬스 밴드, 무선 이어폰                     |

## 11. 임베디드 시스템 사례

### 11.1 인터넷을 사용할 때 거치는 임베디드 장비

인터넷을 사용할 때는 여러 임베디드 장비를 거친다.

```text
스마트폰/컴퓨터
→ Wi-Fi 공유기
→ 모뎀
→ 인터넷 서비스 제공자 장비
→ 초고속망 장비
→ 서버
```

대표 장비는 공유기, 스위치, 모뎀, 기지국 장비이다.
이 장비들은 데이터를 목적지까지 전달하고, 네트워크 연결을 유지하는 역할을 한다.

---

### 11.2 교통수단에서 사용하는 임베디드 시스템

| 교통수단 | 임베디드 시스템 사례                         |
| ---- | ----------------------------------- |
| 버스   | 승하차 단말기, 위치 추적 장치, 전광판, 운행 기록 장치    |
| 지하철  | 개찰구, 교통카드 단말기, 열차 제어 시스템, 안내 방송 시스템 |
| 택시   | 택시 미터기, 카드 결제 단말기, 내비게이션, 블랙박스      |
| 자동차  | ECU, ABS, 에어백 제어기, 주차 보조 시스템, ADAS  |

교통수단에서는 요금 결제, 위치 안내, 안전 제어, 운행 기록을 위해 여러 임베디드 시스템이 사용된다.

---

### 11.3 가정생활에서 사용하는 임베디드 제품

| 제품      | 임베디드 기능                   |
| ------- | ------------------------- |
| 전자레인지   | 타이머, 출력 제어, 버튼 입력 처리      |
| 세탁기     | 세탁 코스 제어, 물 높이 감지, 모터 제어  |
| 냉장고     | 온도 센서, 압축기 제어, 에너지 관리     |
| 스마트 도어락 | 비밀번호 입력, 카드 인식, 잠금 제어     |
| 로봇청소기   | 센서 기반 위치 인식, 모터 제어, 경로 계획 |

---

### 11.4 회사나 학교에서 사용하는 임베디드 장비

| 장비       | 역할                 |
| -------- | ------------------ |
| 출입통제 시스템 | ID카드 또는 지문으로 출입 관리 |
| CCTV     | 보안 영상 촬영 및 저장      |
| 프린터/복합기  | 문서 출력, 스캔, 네트워크 연결 |
| 회의실 장비   | 디스플레이, 화상회의, 음향 제어 |
| 네트워크 장비  | 인터넷 연결 및 내부망 관리    |

---

### 11.5 산업용 로봇

산업용 로봇은 모터, 감속기, 엔코더, 제어기, 센서, 통신 장치로 구성된다.
정해진 작업을 반복적으로 수행하며, 생산성과 정밀도를 높이는 역할을 한다.

대표적인 기능은 다음과 같다.

| 기능       | 설명                        |
| -------- | ------------------------- |
| 위치 제어    | 로봇 팔의 각 관절 위치를 제어한다.      |
| 속도 제어    | 작업 속도를 일정하게 유지한다.         |
| 센서 입력 처리 | 엔코더, 힘 센서, 비전 센서 값을 처리한다. |
| 통신       | PLC나 상위 제어기와 데이터를 주고받는다.  |
| 안전 제어    | 비상 정지, 충돌 방지 기능을 수행한다.    |

---

### 11.6 배송용 드론

배송용 드론은 비행 제어기, GPS, IMU, 카메라, 모터 제어기, 통신 모듈을 포함한다.
위치 제어와 경로 제어가 중요하며, 장애물 회피와 배터리 관리 기능도 필요하다.

| 구성 요소  | 역할                   |
| ------ | -------------------- |
| 비행 제어기 | 드론의 자세와 비행 경로를 제어한다. |
| GPS    | 위치 정보를 제공한다.         |
| IMU    | 가속도와 자세 정보를 측정한다.    |
| 카메라    | 장애물 또는 착륙 지점을 인식한다.  |
| 모터 제어기 | 프로펠러 속도를 제어한다.       |
| 통신 모듈  | 조종기나 서버와 데이터를 주고받는다. |

## 12. 새롭게 만들고 싶은 임베디드 시스템 구상

### 12.1 제품명

**개인 물건 분실 방지 및 외출 준비 확인 시스템**

---

### 12.2 필요성

일상생활에서 지갑, 열쇠, 학생증, 노트북 가방, 이어폰 등을 자주 잃어버리거나 어디에 두었는지 기억하지 못하는 경우가 있다.
기존의 분실 방지 태그는 단순히 위치를 알려주는 수준이지만, 실제 생활에서는 사용자의 외출 상황을 판단하고 필요한 물건을 미리 알려주는 기능이 있으면 더 유용하다.

---

### 12.3 주요 기능

| 기능       | 설명                                |
| -------- | --------------------------------- |
| 위치 추적    | BLE, UWB, GPS 등을 활용해 물건 위치를 확인한다. |
| 거리 알림    | 사용자가 물건에서 멀어지면 알림을 제공한다.          |
| 사용 패턴 학습 | 자주 두는 장소와 시간대를 학습한다.              |
| 외출 전 확인  | 집을 나가기 전 필요한 물건이 있는지 확인한다.        |
| 스마트폰 연동  | 앱을 통해 물건 위치와 상태를 확인한다.            |
| 저전력 동작   | 배터리를 오래 사용할 수 있도록 설계한다.           |

---

### 12.4 필요한 구체적 요소

| 요소             | 설명               |
| -------------- | ---------------- |
| 마이크로컨트롤러       | 센서 데이터 처리와 통신 제어 |
| BLE/UWB 통신 모듈  | 스마트폰과 근거리 통신     |
| 배터리            | 휴대 가능한 전원 공급     |
| 부저 또는 LED      | 근거리에서 위치 알림      |
| 스마트폰 앱         | 물건 위치 확인 및 알림 설정 |
| 센서 데이터 처리 알고리즘 | 위치, 거리, 사용 패턴 분석 |
| 저전력 펌웨어        | 배터리 소모 최소화       |

---

### 12.5 동작 흐름

```text
물건에 스마트 태그 부착
→ 스마트폰과 BLE/UWB 연결
→ 거리 및 위치 정보 측정
→ 사용자의 이동 패턴 분석
→ 외출 상황 판단
→ 필요한 물건이 없으면 알림 발생

```

### 12.6 기대 효과

1. 분실물 발생을 줄일 수 있다.
2. 외출 전 필요한 물건을 자동으로 확인할 수 있다.
3. 스마트폰과 연동하여 생활 편의성을 높일 수 있다.
4. 학생, 직장인, 고령자 등 다양한 사용자에게 도움이 될 수 있다.

## 14. 정리

임베디드 시스템은 스마트폰, 자동차, 가전제품, 네트워크 장비, 로봇, 의료기기 등 일상과 산업 전반에 사용된다.
특히 스마트폰은 가장 많이 사용되는 임베디드 시스템이며, 개인 생활에 가장 큰 영향을 주는 장비이다.

임베디드 시스템은 단순한 전자제품이 아니라 센서 입력, 데이터 처리, 제어 출력이 결합된 시스템이다.
앞으로는 인공지능, 통신 기술, 저전력 기술이 결합되면서 더 많은 장비가 지능화된 임베디드 시스템으로 발전할 것이다.
