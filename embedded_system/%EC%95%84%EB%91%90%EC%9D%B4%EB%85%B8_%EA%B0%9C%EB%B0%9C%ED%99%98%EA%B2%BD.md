# 아두이노 개발환경

## 1. Arduino IDE를 다운로드하고 설치하기
- [공식 Arduino IDE 다운로드 페이지](https://docs.arduino.cc/software/ide-v2/tutorials/getting-started-ide-v2/)에 접속한다.
- 사용 중인 운영체제에 맞는 Arduino IDE를 선택해 다운로드한다.
- 설치 파일을 실행하고 설치 지침에 따라 설치를 완료한다.

## 2. 아두이노 보드를 Arduino IDE에 연결하기
- 아두이노 보드를 USB 케이블로 컴퓨터에 연결한다.
- Arduino IDE를 실행하고 `도구 > 보드`에서 사용하는 보드(예: Arduino Uno)를 선택한다.

## 3. 새로운 아두이노 프로젝트 시작하기
- `파일 > 새로 만들기`를 클릭해 새 프로젝트를 생성한다.
- 기본 코드 템플릿 또는 예제 코드를 불러와 시작하거나, 직접 코드를 작성한다.

## 4. 스케치(프로그램)를 아두이노 보드에 업로드하기
- 상단의 체크 아이콘으로 코드를 검증한 뒤, 화살표 아이콘(업로드 버튼)을 클릭하여 코드를 업로드한다.
- 오류가 발생하면 IDE 하단 메시지를 통해 문제를 확인하고 수정한다.

## 5. 라이브러리를 아두이노 프로젝트에 추가하기
- `스케치 > 라이브러리 포함하기`를 클릭하고 필요한 라이브러리를 선택한다.
- `라이브러리 관리자`를 통해 새로운 라이브러리를 검색하여 설치할 수 있다.

## 6. 아두이노 보드의 포트 선택하기
- `도구 > 포트` 메뉴에서 연결된 아두이노 보드의 COM 포트를 선택한다.
- 올바른 포트가 설정되지 않으면 업로드가 실패할 수 있다.

## 7. 시리얼 모니터로 데이터 모니터링하기
- `도구 > 시리얼 모니터`를 클릭하여 아두이노가 출력하는 데이터를 실시간으로 확인한다.
- `Serial.begin(9600);` 등의 코드가 아두이노 측에 포함되어 있어야 한다.

## 8. 코드 저장 및 프로젝트 관리하기
- `파일 > 저장`을 클릭하여 현재 프로젝트를 저장한다.
- `새 창에서 열기`를 통해 여러 프로젝트를 동시에 관리할 수 있다.

## 9. Arduino IDE의 기본 설정 변경하기
- `파일 > 환경 설정`을 통해 기본 설정을 조정할 수 있다.
- 편집기 글꼴, 출력 메시지 수준, 스케치 저장 경로 등 다양한 항목을 변경할 수 있다.
